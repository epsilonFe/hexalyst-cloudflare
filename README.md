# Hexalyst — Cloudflare Pages

This repository contains the static site sources for hexalyst.com and hexalyst.org,
built for deployment to Cloudflare Pages.

## Structure
- `hexalyst.com/public` — production output for hexalyst.com
- `hexalyst.org/public` — production output for hexalyst.org

## Deploy (Cloudflare Pages)
- hexalyst.com: Root directory `hexalyst.com`, Output `hexalyst.com/public`
- hexalyst.org: Root directory `hexalyst.org`, Output `hexalyst.org/public`

## Security & headers
This repo includes `_headers` files to apply strict security headers and a Content-Security-Policy
that allows local `styles.css` and self‑hosted fonts.
